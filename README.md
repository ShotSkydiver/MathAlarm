MathAlarm is a Windows Phone 7/8 app that I wrote back in 2013. It's an alarm clock app that requires you to solve a math problem in order to turn the alarm off. It features multiple options for the level of difficulty of the math problem, a unique UI, and a fully functional Live Tile. This was the first real app I ever wrote, and was really the first proper coding project I completed! As of today, there have been almost 4,000 unique users who have downloaded and used the app.
